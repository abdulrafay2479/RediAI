About RadiAI

Understanding our AI-powered X-ray analysis platform

🩻 What is RadiAI?

RadiAI is an AI-powered X-ray analysis tool that leverages advanced computer vision technology to provide preliminary diagnostic insights from radiological images. It is designed to help medical students, healthcare professionals, and patients better understand X-ray findings.

Our system uses Claude's state-of-the-art vision capabilities to systematically evaluate X-ray images, identify anatomical structures, detect potential abnormalities, and generate structured diagnostic reports with findings, differential diagnoses, and recommendations.

⚙️ How It Works

1
Image Upload

Users upload X-ray images in standard formats (JPG, PNG, WebP). The system validates format and quality.

2
AI Vision Analysis

The image is sent to Claude's vision model with specialized radiology prompts. The AI systematically evaluates anatomical structures and identifies findings.

3
Report Generation

Results are returned as a structured JSON report with findings, differentials, severity, urgency, and recommendations — all rendered in a clear, professional interface.

Each type uses specialized analysis protocols focusing on relevant anatomical structures and common pathologies.

🔒 Privacy Notice

  • No Image Storage: Uploaded X-ray images are processed in memory and immediately deleted after analysis. No images are permanently stored on our servers.
  • Local History: Analysis history is stored only in your browser's local storage. It never leaves your device.
  • API Processing: Images are sent to the Claude Vision API for analysis. Please review Anthropic's privacy policy for details on how they handle data.
  • No PII Required: Patient context fields (age, sex, symptoms) are optional and are only used to improve analysis accuracy. No personally identifiable information is stored.
